Java

Java 是一种广泛使用的计算机编程语言，拥有跨平台、面向对象、泛型编程的特性，广泛应用于企业级 Web 应用开发和移动应用开发。

Subsections of Java

JDK 新特性

从 JDK8 开始，Java 做了非常大的改进，这反映出 Java 不是一个保守禁锢的语言，而是一个适应时代变化、进化的语言，去努力吸收其他语言的优秀特性。
